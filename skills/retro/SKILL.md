# Retro

Run a candid retrospective on the current session, then update any memory or instruction files that should change as a result.

Answer both prompts below directly and honestly. The goal is genuine improvement, not reassurance.

## Reflect

Based on this session, what went well and what can be improved for next time? Was there anything the user missed? Are there ways that the user can work more effectively with the agent? Don't hold back or be nice. Provide constructive feedback.

## Update

Are any documents outdated and require updating? Is there anything the agent missed or needed to be correct on that should be captured as a memory or in a CLAUDE.md file?
